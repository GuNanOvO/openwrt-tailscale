# LuCI 管理界面

推荐安装由 [@Tokisaki-Galaxy](https://github.com/Tokisaki-Galaxy) 开发的 LuCI 应用来图形化管理 Tailscale。

## luci-app-tailscale-community

这是一个开源的 LuCI 应用，提供了直观的 Web 界面来配置和管理 Tailscale。

- **仓库**: [Tokisaki-Galaxy/luci-app-tailscale-community](https://github.com/Tokisaki-Galaxy/luci-app-tailscale-community)

## 功能

- 查看 Tailscale 状态和连接信息
- 管理 Tailscale 设置
- 启动/停止/重启 Tailscale 服务
- 监控在线节点
- 配置路由和出口节点

## 安装

按照 [luci-app-tailscale-community 仓库](https://github.com/Tokisaki-Galaxy/luci-app-tailscale-community) 的说明进行安装。

::: tip
此 LuCI 应用与本仓库的精简版 Tailscale 完全兼容。
:::
